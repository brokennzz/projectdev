 🚀 ProjectDeV - Responsive Bootstrap 5 Landing Page

ProjectDeV is a responsive web application developed as part of a Software Engineering academic project. The main objective of this project is to demonstrate the implementation of Bootstrap 5's Grid System, responsive design principles, and interactive JavaScript features in a modern user interface.

 📖 Project Overview

This project was created to explore and apply the following concepts:

* Bootstrap 5 Grid System
* Responsive Web Design
* Mobile-First Development
* HTML5 Semantic Structure
* CSS3 Styling and Animations
* JavaScript Interactivity
* User Experience (UX) Best Practices

The website automatically adapts to different screen sizes, providing an optimized experience for mobile devices, tablets, and desktop computers.

---

 ✨ Features

 Responsive Layout

* 1 column on small screens (mobile)
* 2 columns on medium screens (tablet)
* 3 columns on large screens (desktop)

 Modern Interface

* Responsive navigation bar
* Hero section with animated typing effect
* Interactive cards with hover animations
* Smooth scrolling navigation
* Professional visual design

 JavaScript Interactions

* Animated typing text effect
* Animated counters
* Scroll reveal animations
* Bootstrap modal window
* Dark/Light theme switcher

 Bootstrap Components

* Navbar
* Cards
* Buttons
* Badges
* Modal
* Grid System
* Utility Classes
